## Usage:

Compiles and run C, C++, Java and Python files.

```
kp <file>
```

Use ```kp -h``` for help.

## Installation:

Run ```install.sh``` with

```sudo sh install.sh```

_OBS:_ install.sh will install python3-pip and termcolor, cause it is needed for the file.


###### Program uses Python3, tested and used in Ubuntu 20.04 LTS
